# 🧬 English Lab — Application de flashcards

Application PWA pour apprendre l'anglais scientifique (laboratoire, biomécanique, phrases quotidiennes).

## 📱 Installer sur ton smartphone

### Via GitHub Pages (recommandé)

1. **Crée un compte GitHub** sur https://github.com si tu n'en as pas
2. **Crée un nouveau dépôt** (bouton vert "New repository")
   - Nom : `english-lab` (ou ce que tu veux)
   - Coche "Public"
3. **Upload les fichiers** (bouton "uploading an existing file") :
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - Dossier `icons/` avec `icon-192.png` et `icon-512.png`
4. **Active GitHub Pages** :
   - Va dans Settings → Pages
   - Source : "Deploy from a branch"
   - Branch : `main` / dossier `/ (root)`
   - Clique Save
5. Ton app sera disponible sur : `https://TON_PSEUDO.github.io/english-lab`

### Installer sur Android
- Ouvre l'URL dans Chrome
- Menu ⋮ → "Ajouter à l'écran d'accueil"

### Installer sur iPhone
- Ouvre l'URL dans Safari
- Bouton partage (carré avec flèche) → "Sur l'écran d'accueil"

## 🃏 Contenu inclus (40 mots/phrases)

- 🔬 **Laboratoire** (10) : sample, results, measurement, data, protocol...
- 💬 **Quotidien** (10) : by the way, deadline, feedback, actually...
- 🗣️ **Phrases clés** (8) : I'm not sure I follow, Could you slow down...
- 🦴 **Biomécanique** (12) : gait, joint, force plate, motion capture...

## ✨ Fonctionnalités

- Flashcards avec retournement animé
- Système de mémorisation (3 niveaux de maîtrise)
- Phrases d'exemple en anglais + traduction
- Prononciation phonétique
- Compteur de série quotidienne 🔥
- Fonctionne hors-ligne une fois installé
- Sauvegarde locale (reste sur ton téléphone)
